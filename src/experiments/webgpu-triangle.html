WebGPU Triangle

January 28, 2026

This experiment is a little bit like the "hello world" of experiments. It's not so complex, but it allows me to set up WebGPU properly in TypeScript. I'm mostly following along the beginning of this tutorial from Google and adding 3D logic like a camera. Writing precise instructions here would not serve that much purpose so you can check the tutorial out if you want to learn WebGPU. It's a very nice API.

Here is a canvas, if everything goes well you should see a very beautiful triangle!

It's rendered in 3D with Z=0 and a camera at position (0,0,-4) looking directly at (0,0,0)
